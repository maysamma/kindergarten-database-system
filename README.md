# Kindergarten Database System

## Project Overview

A complete database system designed for a kindergarten management environment.

The project covers the full database development lifecycle including:

- Business Rules Analysis
- ER Diagram Design
- UML Modeling
- Relational Mapping
- Database Normalization
- SQL Implementation
- Data Insertion
- Query Development

## Features

- Child Management
- Guardian Management
- Teacher Management
- Administrator Management
- Class Management
- Attendance Tracking

## Technologies

- MySQL
- SQL
- Database Design
- UML
- ER Modeling

## Database Design

The database includes:

- Children
- Guardians
- Teachers
- Principal
- Administrators
- Classes
- Attendance

## Sample Queries

- Count boys and girls
- Retrieve children by class
- Display class teachers
- Find guardians by income
- Display children with allergies
